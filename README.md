# 🚀 Projeto: Análise de Evasão de Clientes (Churn) - Telecom X

Este projeto foi desenvolvido como parte de um desafio de Data Science para identificar padrões e fatores que levam ao cancelamento de clientes na empresa **Telecom X**.

## 📋 Objetivo
Realizar o processo de **ETL (Extração, Transformação e Carga)**, **Análise Exploratória de Dados (EDA)** e o desenvolvimento de **Modelos Preditivos** para antecipar o cancelamento de clientes.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Pandas & Numpy**: Manipulação e tratamento de dados.
- **Requests**: Coleta de dados via API.
- **Matplotlib & Seaborn**: Visualização de dados.
- **Scikit-Learn**: Pré-processamento, normalização e Machine Learning.

## 📂 Estrutura do Projeto
O notebook está organizado nas seguintes etapas:
1. **Extração**: Consumo dos dados brutos em formato JSON.
2. **Transformação**: Limpeza, achatamento de colunas aninhadas e padronização da variável alvo.
3. **Análise Exploratória**: Insights visuais sobre o comportamento do cliente.
4. **Modelagem Preditiva**: Treinamento de modelos (Regressão Logística e Random Forest).
5. **Avaliação**: Análise de métricas (Precisão, Recall e F1-Score) e importância das variáveis.

## 🚀 Como Executar
1. Execute as células em ordem sequencial.
2. O arquivo `dados_tratados.csv` será gerado automaticamente para alimentar os modelos.
3. Os resultados da modelagem e o relatório final fornecerão os insights estratégicos necessários.
